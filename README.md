# Space AI Academy

Браузерная образовательная игра о том, как модели ИИ учатся на примерах,
ошибаются, оценивают уверенность и улучшаются с помощью человека.

## Запуск

Откройте `index.html` в современном браузере. Сборка, сервер и установка
зависимостей не требуются.

## Структура

```text
index.html          — семантическая оболочка приложения
css/tokens.css      — цвета, размеры и визуальные темы глав
css/base.css        — сброс, типографика, фон и общие анимации
css/layout.css      — верхняя панель и композиция экранов
css/ui.css          — кнопки, карточки, робот и разбор ошибок
css/scenes.css      — игровые сцены и финальная анимация
css/responsive.css  — адаптация для планшетов и телефонов
js/config.js        — конфигурация, данные и состояние игры
js/engine.js        — UI, звук, сохранения, частицы и SVG-графика
js/free-lab.js      — свободное обучение и проверка модели на новых данных
js/scenes.js        — главы, игровые механики и переходы
```

Прогресс сохраняется локально в `localStorage`. Звуки генерируются через
Web Audio API, а сертификат — через Canvas API.
